# Actividad #1 - Transparencia en Sistemas Distribuidos
**Fecha:** 29 de abril de 2026

## Descripción
Implementación de transparencia en sistemas distribuidos mediante un sistema de nodos que permite almacenar y buscar archivos de forma distribuida. Demuestra cómo un usuario puede acceder a recursos sin conocer su ubicación física en el sistema.

## Tema
- Transparencia de ubicación
- Búsqueda distribuida de archivos
- Registro de operaciones (logs)

## Archivos
- `practica1_distribuidos.py` - Implementación principal del sistema de nodos
- `A1_codigo_base.py` - Código base de referencia

## Ejecución
```bash
python practica1_distribuidos.py
```

El programa simula múltiples nodos distribuidos que almacenan archivos y permite buscar archivos en todos los nodos del sistema de forma transparente.
